Виселица (угадай слово):
Описание: Игрок пытается угадать слово, предложенное
компьютером, по буквам. За каждую ошибку игроку начисляется
штраф, и он приближается к проигрышу. Особенности реализации: Список или массив для хранения
состояния угаданного слова, функции для проверки букв и
обновления состояния, обработка ввода пользователя. Основные элементы: Управление строками, условные операторы для
проверки попаданий и промахов
